# Mariani Village — cadrage actuel

**Mariani Village** est une initiative d’habitat minimaliste autonome à base de containers maritimes, destinée à loger des étudiants pendant l’année universitaire et des touristes ou occupants saisonniers pendant la saison estivale.

Le README historique du dépôt conserve des publications et éléments de conception antérieurs. Le présent document sert de cadrage synthétique actuel.

## Définition courte

Mariani Village articule :

- logement étudiant ;
- accueil touristique saisonnier ;
- containers maritimes ;
- habitat minimaliste ;
- autonomie énergétique ;
- mobilité annuelle ;
- installation réversible ;
- infrastructures sobres ;
- solidarité entre revenus touristiques et loyers étudiants.

## Logique opérationnelle

Un même module peut loger un étudiant à Corte pendant l’année universitaire, puis être déplacé vers un site littoral ou touristique pendant l’été, avant de revenir à proximité de l’université.

Cette mobilité n’est pas pensée comme une rotation permanente, mais comme un repositionnement annuel discipliné.
