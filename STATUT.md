# Statut du dépôt `acorsica/marianivillage`

Ce dépôt documente le projet **Mariani Village**.

Mariani Village est une initiative d’habitat minimaliste autonome à base de containers maritimes, destinée à loger des étudiants pendant l’année universitaire et des touristes ou occupants saisonniers pendant la saison estivale.

## Qualification actuelle

Ce dépôt est qualifié comme :

> dépôt de projet territorial, social, écologique et technique relatif à l’habitat minimaliste autonome étudiants / touristes.

Il documente une initiative portée historiquement par C.O.R.S.I.C.A. à travers l’Institut Mariani.

## Document source de référence

Le document conceptuel et technique le plus structurant actuellement identifié est :

- [`JeanHuguesRobert/FractaVolta/mariani_village.md`](https://github.com/JeanHuguesRobert/FractaVolta/blob/main/research/mariani_village.md)

Ce document décrit Mariani Village comme une flotte d’habitat relocalisable fondée sur le container ISO 20 pieds comme unité physique minimale.

## Objet

Mariani Village vise notamment :

- le logement étudiant à Corte pendant l’année universitaire ;
- l’accueil touristique ou saisonnier pendant l’été ;
- l’usage de containers maritimes réemployés ou adaptés ;
- l’habitat minimaliste ;
- la réversibilité d’installation ;
- l’autonomie énergétique ;
- l’adaptation aux terrains disponibles ;
- la mobilité lente, avec un déplacement annuel possible entre sites ;
- l’articulation entre solidarité étudiante et revenus touristiques.

## Principe opérationnel

Le container est pensé comme une unité d’habitation mobile ou relocalisable.

Un même module peut être utilisé par un étudiant à Corte d’octobre à juin, puis être déplacé vers un site littoral ou touristique pendant l’été, avant de revenir à proximité de l’université.

Cette mobilité n’est pas une rotation fréquente, mais un repositionnement annuel discipliné.

## Lien avec FractaVolta

Mariani Village entretient un lien fort avec FractaVolta pour les aspects :

- énergie distribuée ;
- autonomie ;
- architecture DC ;
- micro-réseaux ;
- gestion de flotte ;
- infrastructures sobres ;
- transformation énergie / usage / service.

Ce lien est technique, doctrinal et pré-opérationnel.

## Lien avec l’Institut Mariani

Le dépôt relève du périmètre historique de l’Institut Mariani, émanation recherche et développement de C.O.R.S.I.C.A.

Il peut être compris comme une expérimentation autour :

- du logement étudiant ;
- de l’habitat réversible ;
- de l’autonomie énergétique ;
- de la mobilité ;
- des infrastructures sobres ;
- de l’articulation entre tourisme et solidarité étudiante ;
- de la capacité territoriale.

## Lien avec Barons Mariani et MareNostrum

Mariani Village peut entretenir des liens doctrinaux ou techniques avec :

- le futur fonds de dotation Barons Mariani et le Musée Mariani des Possibles ;
- MareNostrum, pour les aspects énergie solaire, compute et autonomie de capacité ;
- FractaVolta, pour les aspects énergie, réseaux, autonomie et infrastructures sobres.

Ces liens doivent être documentés sans confusion de régime.

Ils ne valent ni fusion institutionnelle, ni transfert de propriété, ni portage juridique automatique, ni financement, ni exploitation commerciale, ni soutien politique ou électoral.

> L’interconnexion documentaire ne vaut pas confusion institutionnelle.
